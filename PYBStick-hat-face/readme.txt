# Interface HAT pour PYBStick

La carte d'extension PYBStick-Hat-Face permet d'interfacer facilement un ou deux HAT Raspberry-Pi sur la [PYBStick](https://shop.mchobby.be/fr/recherche?controller=search&orderby=position&orderway=desc&search_query=pybstick&submit_search=).

Branchez votre PYBStick, enfichez votre HAT et c'est parti. Vous pouvez contrôler votre HAT à partir de la PYBStick.
